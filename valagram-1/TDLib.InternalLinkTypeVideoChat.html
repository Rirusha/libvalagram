InternalLinkTypeVideoChat

Object Hierarchy:

Description:

public class InternalLinkTypeVideoChat : InternalLinkType

The link is a link to a video chat. Call Client.search_public_chat with the given chat username, and then Client.join_video_chat with the given invite hash to process the link
