public class GiveawayInfoCompleted : GiveawayInfo

Describes a completed giveaway


Namespace: TDLib
Package: valagram-1

Content:

Properties:

  • public int32 activation_count { get; set construct; }
    Number of winners, which activated their gift codes; for Telegram Premium giveaways only
  • public int32 actual_winners_selection_date { get; set construct; }
    Point in time (Unix timestamp) when the winners were selected. May be bigger than winners selection date specified in parameters of the giveaway
  • public int32 creation_date { get; set construct; }
    Point in time (Unix timestamp) when the giveaway was created
  • public string gift_code { get; set construct; }
    Telegram Premium gift code that was received by the current user; empty if the user isn't a winner in the giveaway or the giveaway isn't a Telegram Premium giveaway
  • public bool is_winner { get; set construct; }
    True, if the current user is a winner of the giveaway
  • public bool was_refunded { get; set construct; }
    True, if the giveaway was canceled and was fully refunded
  • public int32 winner_count { get; set construct; }
    Number of winners in the giveaway
  • public int64 won_star_count { get; set construct; }
    The amount of Telegram Stars won by the current user; 0 if the user isn't a winner in the giveaway or the giveaway isn't a Telegram Star giveaway
