Description:

public class MessageProperties : Error

Contains properties of a message and describes actions that can be done with the message right now


Namespace: TDLib
Package: valagram-1

Properties:

  • public bool can_add_offer { get; set construct; }
    True, if an offer can be added to the message using Client.add_offer
  • public bool can_add_tasks { get; set construct; }
    True, if tasks can be added to the message's checklist using Client.add_checklist_tasks if the current user has Telegram Premium subscription
  • public bool can_be_approved { get; set construct; }
    True, if the message is a suggested post that can be approved by the user using Client.approve_suggested_post
  • public bool can_be_copied { get; set construct; }
    True, if content of the message can be copied using inputMessageForwarded or Client.forward_messages with copy options
  • public bool can_be_copied_to_secret_chat { get; set construct; }
    True, if content of the message can be copied to a secret chat using inputMessageForwarded or Client.forward_messages with copy options
  • public bool can_be_declined { get; set construct; }
    True, if the message is a suggested post that can be declined by the user using Client.decline_suggested_post
  • public bool can_be_deleted_for_all_users { get; set construct; }
    True, if the message can be deleted for all users using the method Client.delete_messages with revoke == true
  • public bool can_be_deleted_only_for_self { get; set construct; }
    True, if the message can be deleted only for the current user while other users will continue to see it using the method Client.delete_messages with revoke == false
  • public bool can_be_edited { get; set construct; }
    True, if the message can be edited using the methods Client.edit_message_text Client.edit_message_caption or Client.edit_message_reply_markup For live location, poll, and checklist messages this fields shows whether Client.edit_message_live_location Client.stop_poll or Client.edit_message_checklist respectively can be used with this message
  • public bool can_be_forwarded { get; set construct; }
    True, if the message can be forwarded using inputMessageForwarded or Client.forward_messages without copy options
  • public bool can_be_paid { get; set construct; }

  • public bool can_be_saved { get; set construct; }

  • public bool can_mark_tasks_as_done { get; set construct; }
