GetChatStoryInteractions

Object Hierarchy:

Description:

public class GetChatStoryInteractions : TDObject

Returns interactions with a story posted in a chat. Can be used only if story is posted on behalf of a chat and the user is an administrator in the chat


Namespace: TDLib
Package: valagram-1

Content:

Properties:

  • public int32 limit { get; set construct; }
    The maximum number of story interactions to return
  • public string offset { get; set construct; }
    Offset of the first entry to return as received from the previous request; use empty string to get the first chunk of results
  • public bool prefer_forwards { get; set construct; }
    Pass true to get forwards and reposts first, then reactions, then other views; pass false to get interactions sorted just by interaction date
  • public ReactionType reaction_type { get; set construct; }
    Pass the default heart reaction or a suggested reaction type to receive only interactions with the specified reaction type; pass null to receive all interactions; reactionTypePaid isn't supported
  • public int32 story_id { get; set construct; }
    Story identifier
  • public int64 story_poster_chat_id { get; set construct; }
    The identifier of the poster of the story
