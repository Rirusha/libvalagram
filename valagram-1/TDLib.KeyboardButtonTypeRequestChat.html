KeyboardButtonTypeRequestChat

Object Hierarchy:

Description:

public class KeyboardButtonTypeRequestChat : KeyboardButtonType

A button that requests a chat to be shared by the current user; available only in private chats. Use the method Client.share_chat_with_bot to complete the request
