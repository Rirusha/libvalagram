BanChatMember

Object Hierarchy:

Description:

public class BanChatMember : TDObject

Bans a member in a chat; requires can_restrict_members administrator right. Members can't be banned in private or secret chats. In supergroups and channels, the user will not be able to return to the group on their own using invite links, etc., unless unbanned first


Namespace: TDLib
Package: valagram-1

Content:

Properties:

  • public int32 banned_until_date { get; set construct; }
    Point in time (Unix timestamp) when the user will be unbanned; 0 if never. If the user is banned for more than 366 days or for less than 30 seconds from the current time, the user is considered to be banned forever. Ignored in basic groups and if a chat is banned
  • public int64 chat_id { get; set construct; }
    Chat identifier
  • public MessageSender member_id { get; set construct; }
    Member identifier
  • public bool revoke_messages { get; set construct; }
    Pass true to delete all messages in the chat for the user that is being removed. Always true for supergroups and channels
