UpdateApplicationVerificationRequired

Object Hierarchy:

Description:

public class UpdateApplicationVerificationRequired : Update

A request can't be completed unless application verification is performed; for official mobile applications only. The method Client.set_application_verification_token must be called once the verification is completed or failed

Package: valagram-1
