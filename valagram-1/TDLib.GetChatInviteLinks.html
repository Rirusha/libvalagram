GetChatInviteLinks

Object Hierarchy:

Description:

public class GetChatInviteLinks : TDObject

Returns invite links for a chat created by specified administrator. Requires administrator privileges and can_invite_users right in the chat to get own links and owner privileges to get other links


Namespace: TDLib
Package: valagram-1

Content:

Properties:

  • public int64 chat_id { get; set construct; }
    Chat identifier
  • public int64 creator_user_id { get; set construct; }
    User identifier of a chat administrator. Must be an identifier of the current user for non-owner
  • public bool is_revoked { get; set construct; }
    Pass true if revoked links needs to be returned instead of active or expired
  • public int32 limit { get; set construct; }
    The maximum number of invite links to return; up to 100
  • public int32 offset_date { get; set construct; }
    Creation date of an invite link starting after which to return invite links; use 0 to get results from the beginning
  • public string offset_invite_link { get; set construct; }
    Invite link starting after which to return invite links; use empty string to get results from the beginning
