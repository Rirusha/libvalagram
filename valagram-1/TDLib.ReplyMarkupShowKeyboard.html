public class ReplyMarkupShowKeyboard : ReplyMarkup

Contains a custom keyboard layout to quickly reply to bots


Namespace: TDLib
Package: valagram-1

Properties:

  • public string input_field_placeholder { get; set construct; }
    If non-empty, the placeholder to be shown in the input field when the keyboard is active; 0-64 characters
  • public bool is_persistent { get; set construct; }
    True, if the keyboard is expected to always be shown when the ordinary keyboard is hidden
  • public bool is_personal { get; set construct; }
    True, if the keyboard must automatically be shown to the current user. For outgoing messages, specify true to show the keyboard only for the mentioned users and for the target user of a reply
  • public bool one_time { get; set construct; }
    True, if the application needs to hide the keyboard after use
  • public bool resize_keyboard { get; set construct; }
    True, if the application needs to resize the keyboard vertically
