KeyboardButtonTypeRequestUsers

Object Hierarchy:

Description:

public class KeyboardButtonTypeRequestUsers : KeyboardButtonType

A button that requests users to be shared by the current user; available only in private chats. Use the method Client.share_users_with_bot to complete the request
