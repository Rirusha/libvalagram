AuthorizationStateWaitPassword

Object Hierarchy:

Description:

public class AuthorizationStateWaitPassword : AuthorizationState

The user has been authorized, but needs to enter a 2-step verification password to start using the application. Call Client.check_authentication_password to provide the password, or Client.request_authentication_password_recovery to recover the password, or Client.delete_account to delete the account after a week
