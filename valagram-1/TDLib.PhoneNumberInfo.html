Properties:

  • public CountryInfo? country { get; set construct; }
    Information about the country to which the phone number belongs; may be null
  • public string country_calling_code { get; set construct; }
    The part of the phone number denoting country calling code or its part
  • public string formatted_phone_number { get; set construct; }
    The phone number without country calling code formatted accordingly to local rules. Expected digits are returned as '-', but even more digits might be entered by the user
  • public bool is_anonymous { get; set construct; }
    True, if the phone number was bought at https://fragment.com and isn't tied to a SIM card. Information about the phone number can be received using Client.get_collectible_item_info
