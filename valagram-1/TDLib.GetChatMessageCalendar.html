GetChatMessageCalendar

Object Hierarchy:

Description:

public class GetChatMessageCalendar : TDObject

Returns information about the next messages of the specified type in the chat split by days. Returns the results in reverse chronological order. Can return partial result for the last returned day. Behavior of this method depends on the value of the option "utc_time_offset"


Namespace: TDLib
Package: valagram-1

Content:

Properties:

  • public int64 chat_id { get; set construct; }
    Identifier of the chat in which to return information about messages
  • public SearchMessagesFilter filter { get; set construct; }
    Filter for message content. Filters searchMessagesFilterEmpty, searchMessagesFilterMention, searchMessagesFilterUnreadMention, and searchMessagesFilterUnreadReaction are unsupported in this function
  • public int64 from_message_id { get; set construct; }
    The message identifier from which to return information about messages; use 0 to get results from the last message
  • public MessageTopic topic_id { get; set construct; }
    Pass topic identifier to get the result only in specific topic; pass null to get the result in all topics; forum topics aren't supported
