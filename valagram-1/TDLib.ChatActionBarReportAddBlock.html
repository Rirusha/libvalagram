public class ChatActionBarReportAddBlock : ChatActionBar

The chat is a private or secret chat, which can be reported using the method Client.report_chat or the other user can be blocked using the method Client.set_message_sender_block_list or the other user can be added to the contact list using the method Client.add_contact If the chat is a private chat with a user with an emoji status, then a notice about emoji status usage must be shown
