GetChatMessageCount

Object Hierarchy:

Description:

public class GetChatMessageCount : TDObject

Returns approximate number of messages of the specified type in the chat


Namespace: TDLib
Package: valagram-1

Content:

Properties:

  • public int64 chat_id { get; set construct; }
    Identifier of the chat in which to count messages
  • public SearchMessagesFilter filter { get; set construct; }
    Filter for message content; searchMessagesFilterEmpty is unsupported in this function
  • public bool return_local { get; set construct; }
    Pass true to get the number of messages without sending network requests, or -1 if the number of messages is unknown locally
  • public int64 saved_messages_topic_id { get; set construct; }
    If not 0, only messages in the specified Saved Messages topic will be counted; pass 0 to count all messages, or for chats other than Saved Messages
